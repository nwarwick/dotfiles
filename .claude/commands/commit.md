## Context

- Current git status: !`git status`
- Current git diff (staged and unstaged changes): !`git diff HEAD`
- Current branch: !`git branch --show-current`
- Recent commits: !`git log --oneline -10`

## Your task

Based on the above changes, create a single git commit.

## Rules

- Match the commit message style of recent commits shown above
- Keep the message concise: what changed and why
- Do NOT include any AI/Claude references, co-author tags, or "Generated by" lines
- If there are no changes to commit, say so and stop
- Only output text if there's an error or ambiguity requiring input

Stage all changes and create the commit in a single message.
